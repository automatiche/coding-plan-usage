# Coding Plan Usage (Minimax)

在 VS Code / Cursor 状态栏显示 Minimax Coding Plan 额度使用情况 | Show Minimax Coding Plan usage in status bar

## 功能 Features

- 在状态栏显示当前周期额度使用百分比
- 悬停查看当前周期时间段和重置时间
- 点击刷新额度

## 安装 Installation

### 从 VS Code 扩展市场安装（推荐）

1. 打开 VS Code / Cursor
2. 进入扩展市场，搜索 `Coding Plan Usage (Minimax)`
3. 点击安装

## 配置 Configuration

1. 打开 VS Code / Cursor
2. `Cmd+,` (Windows: `Ctrl+,`) 打开设置
3. 搜索 `codingPlanUsage.apiKey`
4. 输入你的 Minimax API Key(是Coding Plan 的的 API Key)

## 使用 Usage

- 状态栏左侧显示已使用百分比: `✨ 5%`
- 鼠标悬停查看详情（当前周期、重置时间）
- 点击刷新额度

## 支持 Supported

- 仅支持 **Minimax Coding Plan**
- 兼容 VS Code 和 Cursor

## 获取 API Key

从 [Minimax 开放平台](https://platform.minimaxi.com/) 获取 API Key
